# Project 3 - Buddy System Memory Management

## Files
`Project3.java`
This file contains all the source code for the project. The Project3 class has a main method which
has driver code to run the worker-customer simulation.
`input.txt`
This file has the input commands to run on the buddy memory system.

## Compiling & Running
The program can be compiled using the following command:
`javac Project3.java`
Then, the project can be run as:
`java Project3`
This commands can be run together if desired with the following command:
`javac Project3.java && java Project3`
